This project was done for the class ‘Designing Communications for Interactions.’ We each were assigned a heteronym: a set of two different words that are spelled the same but have different sounds and meanings. I created an interaction for the heteronym 'close' with the goal of communicating the meaning and the feeling of the word only using letters, motion, and sound. Click each word below to try the interaction yourself!

close (adj.): being in or having proximity in space or time

close (verb): to stop or obstruct (a gap, entrance, aperture, etc.)
